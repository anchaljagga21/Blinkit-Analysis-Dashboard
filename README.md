# Blinkit-Analysis-Dashboard
This repository contains a comprehensive **Blinkit Analysis Dashboard** designed using **Power BI**. 
The dashboard examines store performance, sales distribution, customer ratings, pricing trends, 
and item availability. It delivers data-driven insights useful for retail and quick-commerce analysis.

**Project Objectives**
- Analyze **Sales**, **Demand**, and **Ratings Trends**
- Understand performance across **Store Types** and **Cities**
- Compare **Category-wise Sales & Item Availability**
- Identify customer engagement and satisfaction patterns
